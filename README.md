# Health Vault

Simple beginner-friendly Flask app where:

- Patients can register, log in, and upload medical reports.
- Patients can generate temporary access keys to share reports.
- Doctors can use access keys to view patient reports.

This project uses:

- Python Flask
- SQLite
- HTML + CSS (no JS frameworks)

To run (after Python and pip are installed):

1. Create and activate a virtual environment (recommended).
2. Install dependencies:
   - `pip install -r requirements.txt`
3. Run the app:
   - `python app.py`
4. Open `http://127.0.0.1:5000` in your browser.
